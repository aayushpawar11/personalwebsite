# Public Assets

## Adding Your Resume

1. Place your resume PDF file in this folder (`public/`)
2. Name it exactly: `resume.pdf`
3. The resume button on the website will automatically link to it

**Note:** The file must be named `resume.pdf` for the button to work correctly.

## Adding Your Headshot

1. Place your headshot photo in this folder (`public/`)
2. Name it exactly: `headshot.jpg` (or `.png` if it's a PNG file)
3. The image will automatically appear in the hero section

**Supported formats:** JPG, JPEG, PNG
**Recommended size:** Square image (e.g., 800x800px or larger) for best quality
**Note:** If using PNG, update the file extension in `src/components/Hero.jsx` (line with `src="/headshot.jpg"`)
